## ✨ Features

- 🖱 **Drag & drop** or click to upload — PNG, JPG, WebP, GIF
- 📐 **Custom dimensions** with aspect ratio lock
- 🎯 **9 built-in presets** — HD, FHD, 4K, OG Image, Instagram Square, Story, Favicon & more
- 🔄 **3 fit modes** — Contain (letterbox), Cover (crop), Stretch
- 🎚 **Quality control** for JPEG & WebP output
- 💾 **Export as** JPEG, PNG, or WebP
- 📋 **Copy to clipboard** in one click
- 📊 **Live file size estimate** before downloading
- ✅ **100% client-side** — your images never leave your browser

## 🛠 Run Locally

No install needed. Just clone and open:

```bash
git clone https://github.com/starcoder346/image-resizer.git
cd image-resizer
open index.html   # or just double-click the file
```
